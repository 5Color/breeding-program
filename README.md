# breeding-program
Genetic Algorithm based Plant Breeding Simulator with Biological Constraints

---

# 생물학적 제약 기반 품종개량 시뮬레이터 (v7)

**유전 알고리즘(Genetic Algorithm)을 활용한 차세대 작물 품종 최적화 도구**

이 프로젝트는 단순한 데이터 연산을 넘어, 실제 생물학적 원리인 **코돈(Codon) 체계와 유전적 제약 조건**을 프로그래밍 로직으로 구현한 시뮬레이터입니다. 인류의 식량 문제를 해결하고자 하는 '제1원리 사고'를 바탕으로 설계되었습니다.

---

## 핵심 기능 (Key Features)

* **유전 알고리즘(GA) 기반 최적화**: 수천 번의 세대 진화(Crossover, Mutation)를 통해 목표 형질(당도, 수확량 등)에 가장 근접한 최적의 DNA 염기서열을 찾아냅니다.
* **실제 생물학적 제약 반영**:
* **코돈(Codon) 단위 연산**: 3개 염기가 하나의 아미노산을 형성하는 생명과학의 기본 원리를 반영하여 시뮬레이션합니다.
* **종결 코돈(Stop Codon) 패널티**: 품종 개량 중 의도치 않게 단백질 합성이 끊기는 현상을 방지하는 로직이 포함되어 있습니다.
* **GC 함량(GC Content) 유지**: DNA의 화학적 안정성을 위해 원본 서열의 GC 비율을 추적하고 패널티를 부여합니다.
* **아미노산 변이 관리**: 원본 대비 아미노산 서열이 과도하게 변하지 않도록 관리하여 생물학적 유효성을 높입니다.


* **다양한 작물 프리셋**: 토마토(당도/숙성기), 고추(매운맛/크기), 쌀(수확량/미질) 등 주요 작물의 형질 가중치 데이터가 내장되어 있습니다.
* **실제 유전체 데이터 호환**: NCBI 등에서 추출한 실제 DNA 염기서열(FASTA 등)을 입력값으로 사용하여 실무적인 연구 시뮬레이션이 가능합니다.

---

## 기술 스택 (Tech Stack)

* **Language**: Python 3
* **Library**: `tqdm` (실시간 진화 프로세스 시각화)
* **Algorithm**: Genetic Algorithm (Selection, Crossover, Mutation)

---

## 사용 방법 (Quick Start)

1. **환경 설정**: 필요한 라이브러리를 설치합니다.
```bash
pip install tqdm

```


2. **프로그램 실행**:
```bash
python breeding-program.py

```


3. **시나리오 설정**:
* 원하는 작물 인덱스(1~3)를 선택합니다.
* 당도, 수확량 등 세부 형질의 목표값을 입력합니다.
* 초기 DNA 서열을 입력하거나 랜덤 생성하여 진화를 시작합니다.
